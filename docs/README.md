# Docs do projeto

Esta pasta guarda briefings, guias de conteúdo e anotações do projeto `ecommerce-syntaxWear`.

Estrutura recomendada:

- `briefing.md` — resumo do projeto e objetivos
- `content-guidelines.md` — instruções para copy, categorias e SEO
- `assets/` — notas sobre imagens e tamanhos

Observações práticas:

- Coloque logos em `images/logo/`.
- Banners em `images/banners/`.
- Produtos organizados por categoria em `images/products/<categoria>/` — crie uma subpasta para cada categoria (ex: `t-shirts`, `hoodies`, `accessories`).
- Ícones SVG em `icons/`.
- Se for usar fontes hospedadas localmente, coloque em `fonts/`.
- Favicons em `favicons/`.

Comece preenchendo `briefing.md` com o público-alvo e prioridades de conteúdo.
